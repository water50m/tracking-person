# บทที่ 2: ทฤษฎีและเทคโนโลยีที่เกี่ยวข้อง

ในการจัดทำโครงงานนี้ ผู้จัดทำได้ศึกษาค้นคว้าทฤษฎี เอกสารวิชาการ และงานวิจัยที่เกี่ยวข้อง เพื่อนำมาเป็นกรอบแนวคิดและประยุกต์ใช้ในการออกแบบระบบ โดยมีรายละเอียดของทฤษฎีและเทคโนโลยีดังต่อไปนี้

## 2.1 ทฤษฎีและหลักการพื้นฐาน (Theoretical Background)
[อธิบายทฤษฎีที่เป็นแกนกลางของโปรเจกต์]

**2.1.1 หลักการทำงานของปัญญาประดิษฐ์และคอมพิวเตอร์วิทัศน์ (AI & Computer Vision)**
- [อธิบายกระบวนการสกัดคุณลักษณะ (Feature Extraction) ของภาพ]
- [อธิบายหลักการของ Object Detection และ Object Tracking]

**2.1.2 ทฤษฎีทางคณิตศาสตร์และสมการที่เกี่ยวข้อง**
[อธิบายสมการที่ใช้ในอัลกอริทึม หรือการคำนวณทางวิศวกรรม]
ตัวอย่างการคำนวณระยะทางแบบยุคลิด (Euclidean Distance):
$$d(p, q) = \sqrt{\sum_{i=1}^{n} (q_i - p_i)^2}$$

**2.1.3 ทฤษฎีการประเมินประสิทธิภาพ (Evaluation Metrics)**
[สำคัญมากสำหรับงาน AI/System ต้องบอกว่าทฤษฎีการวัดผลคืออะไร]
- **Accuracy (ความแม่นยำ):** อัตราส่วนของการทำนายที่ถูกต้องทั้งหมด
- **Precision และ Recall:** สำหรับประเมินความถูกต้องของการตรวจจับ
- **FPS (Frames Per Second):** ทฤษฎีการวัดความเร็วในการประมวลผลภาพแบบเรียลไทม์

[อธิบายว่าเครื่องมือแต่ละตัวคืออะไร มีข้อดีอย่างไร ทำไมถึงเลือกใช้]

**2.2.1 ภาษาโปรแกรมและเฟรมเวิร์ก (Programming Languages & Frameworks)**

- **Next.js & Tailwind CSS:** [อธิบายการใช้สร้าง Web Application และ UI/UX]
- **Python:** [อธิบายการใช้เป็นภาษาหลักในการรันโมเดล Deep Learning]

**2.2.2 ไลบรารีและสถาปัตยกรรมโมเดล (Libraries & Model Architectures)**
- **YOLO (You Only Look Once):** [อธิบายสถาปัตยกรรมเครือข่ายประสาทเทียม (CNN) ของ YOLO]
- **DeepSORT (ถ้ามี):** [อธิบายอัลกอริทึมสำหรับการติดตามวัตถุ]

**2.2.3 ฮาร์ดแวร์และอุปกรณ์ระบบ (Hardware & System Components) (ถ้ามี)**
