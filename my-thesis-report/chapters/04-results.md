## 4.3 ผลการทดสอบและประเมินประสิทธิภาพ (Evaluation Results)
[ส่วนนี้สำคัญมาก! ต้องมีตัวเลขหรือกราฟมายืนยันว่าระบบทำงานได้ดีแค่ไหน]

**4.3.1 ผลการทดสอบความแม่นยำของโมเดล (Model Accuracy) (ถ้ามี AI)**
ใช้สมการการหาค่า Precision และ Recall ในการประเมินผล:
$$Precision = \frac{TP}{TP + FP}$$
$$Recall = \frac{TP}{TP + FN}$$

**ตารางที่ 4.1** ผลการประเมินความแม่นยำของการตรวจจับวัตถุ
| คลาสที่ตรวจจับ (Class) | Precision | Recall | mAP@0.5 |
| :--- | :---: | :---: | :---: |
| วัตถุประเภท A | 0.92 | 0.89 | 0.94 |
| วัตถุประเภท B | 0.85 | 0.82 | 0.88 |
| **เฉลี่ยรวม** | **0.88** | **0.85** | **0.91** |

**4.3.2 ผลการทดสอบประสิทธิภาพของระบบ (System Performance)**
- **ความเร็วในการประมวลผล (Processing Speed):** ระบบสามารถประมวลผลภาพได้ที่ความเร็วเฉลี่ย 30 FPS บนการ์ดจอ...
- **ความเสถียรของ Web Application (Web Performance):** การโหลดหน้าเว็บ (Page Load Time) เฉลี่ยอยู่ที่... วินาที และ API ตอบสนองภายใน... มิลลิวินาที
